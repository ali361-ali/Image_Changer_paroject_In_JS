# Image_Changer_paroject_In_JS

🚀 Features
Dynamic Rendering: Movies are automatically generated from a JSON-like array.

Responsive Design: Image layouts adapt to different screen sizes.

Category Filtering: (Planned) Sort movies by Action, Animation, or Thriller.

Clean UI: Uses high-quality backdrops and posters from TMDB.

🛠️ Tech Stack
HTML5: Structure of the gallery.

CSS3: Styling and layout (Flexbox/Grid).

JavaScript (ES6+): Logic, .map() method, and DOM injection.
